## 專案概述
「柳丁甜度智能預測系統」是一個結合了影像分析和機器學習技術的創新專案，旨在透過分析柳丁的 RGB 影像來預測其甜度。

## 數據處理與模型訓練
1. 數據收集與預處理：首先，收集大量柳丁的 RGB 影像數據。這些數據經過預處理，包括清洗、標準化，以及適當的數據增強，以提高模型訓練的有效性。

2. Python 模型訓練：使用 Python 語言和其豐富的深度學習庫（TensorFlow），開發了一個深度學習模型。這個模型被訓練來識別和學習影像中的甜度相關特徵。

3. 模型優化與評估：經過多次迭代和調整，模型達到了93%的高預測準確率。此外，通過交叉驗證和實際應用測試來進一步驗證模型的穩定性和可靠性。
